# 🎄 Advent of Code — Day 1

Solutions for **Advent of Code Day 1**, implemented in **Rust**.

This project contains:

- A **part 1** solution (`src/bin/day1_part1.rs`)
- A **part 2** solution (`src/bin/day1_part2.rs`)
- A shared `input.txt`
- Unit tests for both parts

---

## ▶️ Running the Solutions

### **Part 1**
```bash
cargo run --bin day1_part1 < input.txt
```
### **Part 2**
```bash
cargo run --bin day1_part2 < input.txt
```

## **🧪 Running Tests**

### **Run all test**
```bash
cargo test
```

### **Part 1**
```bash
cargo test --bin day1_part1
```
### **Part 2**
```bash
cargo test --bin day1_part2
```
